- in making your own website, are you happy with the tools you used?

I will be using Visual Studio for producing my website as it has very useful functionality such as predictive text, function templates, and debugging capabilities. It also runs on browser so one can do work across platforms. Trello was used to plan and track project development. To-do lists are important for the efficient running of a project. They allow timely reminders of the steps that need to be taken and also promote effective time management. 2.2 Describe whether the IT tools selected were appropriate
e.g. Visual Studio Code, Codepen, Trello

I have been very happy with Visual Studio Code as the compiler of choice. Its predictive text is very helpful and time efficient. Also the ability to read and write in many different languages on the same platform, HTML and CSS in particular, is very useful. However, Codepen became a bit redundant during the latter phase of the course because Visual Studio Code is much more of a powerful IDE software, with readability of many coding languages. It can be seen that the application has great power, especially in collaborative roles. I would most definitely use this web-app in future, particularly in large projects with many roles and sections.
